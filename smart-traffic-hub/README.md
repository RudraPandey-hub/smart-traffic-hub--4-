# Smart Traffic Hub

## Quick start
1. run `npm install`
2. create `.env` from `.env.example` and set `VITE_GOOGLE_MAPS_API_KEY`
3. run `npm run dev`

## Deploy to Vercel
- connect repo, set env vars, deploy

## Docker
- build with `docker build -t smart-traffic-hub .`
- run with `docker run -p 80:80 smart-traffic-hub`
